# Focus Timer 🕐

## Overview
A beautiful, vanilla HTML/CSS/JS Pomodoro-style focus timer with modern UX.

## Features
- **Custom Durations**: Presets (Work 25m, Break 5m, Long Break 15m) + custom input (1-60 min).
- **Live Countdown**: MM:SS display with leading zero.
- **Controls**: Start, Pause, Reset.
- **Visual Warnings**: Last 10s → red pulsing timer + "Finishing soon!" alert.
- **Responsive**: Works on desktop/mobile.
- **No Dependencies**: Pure vanilla code, instant browser load.

## Usage
1. Click preset or enter custom minutes.
2. **Start** → countdown begins.
3. **Pause** → stops anytime.
4. **Reset** → reloads selected duration.
5. Browser alert at 00:00.
